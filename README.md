# منصة إدارة حجز القاعات

تطبيق شامل مبني بـ Vue 3 + TypeScript لإدارة حجز القاعات، مبني بأحدث تقنيات الويب.

## 🚀 المميزات

### الوحدات الأساسية

- **إدارة المصادقة والجلسات** - تسجيل دخول آمن مع رموز JWT
- **إدارة القاعات والمرافق** - عمليات CRUD كاملة للقاعات
- **إدارة أنواع الحجوزات** - تكوين أنواع مختلفة من الحجوزات
- **إدارة أنواع الأنشطة** - إدارة الأنشطة المختلفة التي يمكن إجراؤها في القاعات
- **إدارة المستخدمين** - إدارة حسابات المستخدمين مع صلاحيات قائمة على الأدوار

### المميزات الرئيسية

- 🔐 مصادقة آمنة مع حماية المسارات
- 📱 تصميم متجاوب باستخدام Vuetify 3
- 🎨 واجهة مستخدم حديثة مع أيقونات Material Design
- 📊 جداول بيانات مع البحث والتصفية
- 📁 دعم رفع الملفات لصور القاعات والمرفقات
- 🔄 تحديثات الحالة في الوقت الفعلي
- 📱 واجهة صديقة للأجهزة المحمولة

## 🛠️ التقنيات المستخدمة

- **الواجهة الأمامية**: Vue 3 + TypeScript
- **إطار العمل**: Vuetify 3
- **التصميم**: TailwindCSS
- **إدارة الحالة**: Pinia
- **التوجيه**: Vue Router 4
- **عميل HTTP**: Axios
- **أداة البناء**: Vite
- **الأيقونات**: Material Design Icons

## 📋 المتطلبات الأساسية

- Node.js 20.19.0 أو أحدث
- مدير الحزم npm أو yarn

## 🚀 التثبيت

1. **استنساخ المستودع**

   ```bash
   git clone <repository-url>
   cd occasioni
   ```

2. **تثبيت التبعيات**

   ```bash
   npm install
   ```

3. **إنشاء ملف البيئة**
   أنشئ ملف `.env` في المجلد الجذر:

   ```env
   # API Configuration
   VITE_API_BASE_URL=https://occasioni.fexpress.ly/api

   # App Configuration
   VITE_APP_TITLE=نظام إدارة حجز القاعات
   VITE_APP_VERSION=1.0.0

   # Environment
   NODE_ENV=development
   ```

   **ملاحظة مهمة**: تأكد من إنشاء ملف `.env` في المجلد الجذر للمشروع (نفس مستوى `package.json`)

4. **بدء خادم التطوير**

   ```bash
   npm run dev
   ```

5. **افتح متصفحك**
   انتقل إلى `http://localhost:5173` أو `http://localhost:5174` (حسب المنفذ المتاح)

## 🔐 المصادقة

يستخدم التطبيق رموز JWT للمصادقة:

- **تسجيل الدخول**: POST `/api/auth/login`
- **التحقق من الجلسة**: GET `/api/auth/me`
- **حماية المسارات**: إعادة توجيه تلقائية لصفحة تسجيل الدخول للمسارات المحمية
- **تخزين الرمز**: تخزين آمن في localStorage مع تنظيف تلقائي

## 📊 نقاط نهاية API

### المصادقة

- `POST /api/auth/login` - تسجيل دخول المستخدم
- `GET /api/auth/me` - الحصول على معلومات المستخدم الحالي

### القاعات

- `GET /api/hall` - قائمة بجميع القاعات
- `GET /api/hall/{id}` - الحصول على تفاصيل القاعة
- `POST /api/hall` - إنشاء قاعة جديدة
- `PATCH /api/hall/{id}` - تحديث القاعة
- `POST /api/hall/{id}/block` - حظر/إلغاء حظر القاعة
- `POST /api/hall/{id}/hallImage` - رفع صورة القاعة
- `POST /api/hall/{id}/hallAttachments` - رفع المرفقات
- `POST /api/hall/{id}/user` - إضافة مستخدم للقاعة
- `GET /api/hall/{id}/users` - قائمة مستخدمي القاعة
- `POST /api/hall/user/{id}/block` - حظر مستخدم القاعة

### أنواع الحجوزات

- `GET /api/ReservationType` - قائمة بجميع الأنواع
- `POST /api/ReservationType` - إنشاء نوع جديد
- `PATCH /api/ReservationType/{id}` - تحديث النوع
- `POST /api/ReservationType/{id}/block` - حظر/إلغاء حظر النوع

### أنواع الأنشطة

- `GET /api/typeOfActivities` - قائمة بجميع الأنشطة
- `POST /api/typeOfActivities` - إنشاء نشاط جديد
- `PATCH /api/typeOfActivities/{id}` - تحديث النشاط
- `POST /api/typeOfActivities/{id}/block` - حظر/إلغاء حظر النشاط

### المستخدمين

- `GET /api/users` - قائمة بجميع المستخدمين
- `GET /api/users/{id}` - الحصول على تفاصيل المستخدم
- `POST /api/users` - إنشاء مستخدم جديد
- `PATCH /api/users/{id}` - تحديث المستخدم
- `POST /api/users/{id}/block` - حظر/إلغاء حظر المستخدم

## 🎯 الاستخدام

### 1. المصادقة

- انتقل إلى `/login`
- أدخل بيانات الاعتماد الخاصة بك
- بعد تسجيل الدخول بنجاح، سيتم إعادة توجيهك إلى لوحة التحكم

### 2. لوحة التحكم

- عرض إحصائيات النظام
- وصول سريع للميزات الرئيسية
- نظرة عامة على النشاط الأخير

### 3. إدارة القاعات

- عرض جميع القاعات في جدول بيانات
- البحث وتصفية القاعات
- إنشاء وتعديل وحذف القاعات
- رفع الصور والمرفقات
- إدارة مستخدمي القاعة

### 4. إدارة المستخدمين

- إدارة حسابات المستخدمين
- تعيين الأدوار (مدير، مشرف، مستخدم)
- حظر/إلغاء حظر المستخدمين
- عرض إحصائيات المستخدم والحجوزات
